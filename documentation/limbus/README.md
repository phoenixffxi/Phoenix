# 75-cap Limbus

## Purpose

FFXI had a version of Limbus available from its release in CoP up to early 2025 where it was removed for a reworked version for 99 cap.

The files in this folder can be used as a replacement for servers that wish to utilize the existing 75 cap files from LSB prior to the February 2025 version update.

## Utilization

1. Remove existing files in the current Apollyon / Temenos Zone folders
2. Duplicate the files in each zone in this directory to their respective zones
3. Obtain DATs from prior to the 2025 February version and replace Apollyon/Temenos files. (TODO: Instructions for using XIDivert)

   - This will require adjusting the attached IDs.lua file according to the version used which can be obtained through event dumps.
